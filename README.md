# Landing-Page-Turismo
Landing Page de turismo em Recife

Landing Page de turismo desenvolvida utilizando apenas HTML e CSS para o desafio [Local Turístico] para o plano de estudo [2026.1 - Ciclo RiseUp FICR] da Rocketseat . A página apresenta informações com destaque para locais turísticos, imagens atrativas e textos descritivos. O objetivo é praticar a estruturação semântica do HTML e o uso de estilos em CSS para criar um layout organizado e visualmente agradável.
